데이터 관리

localStorage 저장을 유지하고 JSON으로 백업/복원합니다.

이 앱은 로그인 없이 사용하는 로컬 저장 방식입니다.

회원 정보는 서버가 아니라 현재 사용 중인 기기와 브라우저의 localStorage에 저장됩니다.

같은 주소라도 기기/브라우저가 다르면 데이터가 자동 공유되지 않습니다.
  • 아이폰 Safari에 입력한 데이터는 아이폰 Safari에만 남습니다.
  • PC, 다른 휴대폰, 다른 브라우저에서는 JSON 복원이 필요합니다.
  • 브라우저 데이터 삭제 또는 기기 변경 시 데이터가 사라질 수 있습니다.
데이터 보전을 위해 JSON 백업을 꼭 사용해 주세요.
  • JSON 백업: 현재 회원/출석/PT/메모 데이터를 파일로 저장합니다.
  • JSON 복원: 저장해 둔 백업 파일을 다시 불러옵니다.
  • 백업 파일은 iCloud Drive, Google Drive 같은 안전한 곳에 보관하는 것을 권장합니다.
